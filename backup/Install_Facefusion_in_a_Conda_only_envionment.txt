Install Facefusion in a Conda enviornment only

0. Open PowerShell:
   - Open the Start menu, type "PowerShell", right-click on "Windows PowerShell", and select "Run as administrator" (recommended).

1. Install Git:
   winget install -e --id Git.Git

2. Install Conda:
   winget install -e --id Anaconda.Miniconda3
   - Close and reopen PowerShell to ensure Conda is added to the system path.

3. Verify Conda Installation:
   conda --version
   conda info
   check to make sure conda path is set in your path 

4. Activate the base Conda environment and update pip:
   conda create --name facefusion python=3.10
   conda activate facefusion
   python -m pip install --upgrade pip

5. Install Microsoft Visual C++ 2015 Redistributable:
   winget install -e --id Microsoft.VCRedist.2015+.x64

6. Install Microsoft Visual Studio 2022 Build Tools:
   winget install -e --id Microsoft.VisualStudio.2022.BuildTools --override "--wait --add Microsoft.VisualStudio.Workload.NativeDesktop --includeRecommended"

7. Install FFmpeg:
   winget install -e --id Gyan.FFmpeg
   - Reboot your system to ensure FFmpeg functions properly.

8. clone the FaceFusion Repository:
   Navigate to the desired installation location in file explorer and type cmd enter in the address bar 
   cd C:\path\to\root\install\location\
   git clone https://github.com/facefusion/facefusion
   cd facefusion

9. Create a Conda Environment for FaceFusion:
   conda create --name facefusion python=3.10
   conda activate facefusion

10. Install CUDA (if using GPU acceleration):
    conda install -c conda-forge cudatoolkit=11.8 cudnn=8.4.0

11. Install FaceFusion Dependencies:
    python install.py -skip-venv

12. Run FaceFusion:
    python run.py

13. (Optional) Create a Batch File to Run FaceFusion:
    - Create a file named 'facefusion.bat' in the FaceFusion directory with the following contents:
      ```
      @echo off
      call conda activate facefusion
      python run.py
      pause
      ```
    - This batch file can be used to easily run FaceFusion with the correct conda environment activated. Simply double-click 'facefusion.bat' to start FaceFusion.
